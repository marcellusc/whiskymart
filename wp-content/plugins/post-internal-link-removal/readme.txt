This plugin will remove or replace all internal link within a post.

== Description ==

By using this plugin user can remove or replace all internal link which is persent in post.User can also search specific word within a post for putting own link.

== Installation ==

The quickest method for installing the importer is:

1. Upload the plugin folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress


== Frequently Asked Questions ==

= How to use? =

Under Starting Point you have to enter starting number of record from where you want to replace the link and Ending point where you have to mention how many post's inter link you want to remove.

= Can i put my own link? =

Yes, you can put your own link as well.You can mentione your link under Destination link. This link will be used to replace all internal link which is in your post.Only mentione your link like google.com,yourdomain.com etc.

= Can i put link on specific word? =

Yes, You can do this by mentione that word in Search Keyword. But for this you will also have to mentione the post ID. In this case only one post will be used.
